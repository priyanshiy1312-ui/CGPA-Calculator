
# 🎓  CGPA Calculator

A responsive and interactive **CGPA Calculator** built using **HTML, CSS, and JavaScript**. This application allows students to add multiple subjects, enter credits and marks, calculate SGPA for each semester, and track their overall CGPA with ease.

## ✨ Features

* ➕ Add multiple subjects dynamically
* 📝 Enter subject-wise credits and marks
* 📊 Calculate Semester Grade Point Average (SGPA)
* 🎓 Calculate Cumulative Grade Point Average (CGPA)
* 🔄 Reset functionality
* 📱 Responsive and user-friendly interface
* ⚡ Real-time calculation and result display

## 🛠️ Technologies Used

* 🌐 HTML5
* 🎨 CSS3
* ⚡ JavaScript (ES6)

## 🚀 How to Use

1. Add subjects using the **Add Subject** button.
2. Enter subject names, credits, and marks.
3. Click **Calculate SGPA** to calculate the semester SGPA.
4. Save the semester result.
5. Repeat for additional semesters.
6. View the overall CGPA calculated from all saved semesters.
